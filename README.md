# LIS545-Data-Curation-Final-Project
## About
This repository contains files associated with my submission for the final project in LIS 454 - Data Curation I: Fundamentals. The purpose of this project is to demonstrate data curation best practices surrounding file description, metadata, and digital resource organization using a publicly accessible dataset from the FigShare repository.

The dataset used in this project is originally located [here](https://figshare.com/articles/dataset/User_Experience_in_Virtual_Self-Disclosure_Appraising_Natural_Urban_and_Artificial_VR_Environments/30692063?file=59788994) on FigShare and was created as part of a psychological study: "User Experience in Virtual Self-Disclosure: Appraising Natural, Urban, and Artificial VR Environments"

## Original Data Files
The following resources are the original files, in their original format, taken from FigShare. There are six files, three of which contain raw data from the study and the other three which contain codebooks for interpreting the data.<br>

Raw Data:<br>
   * [Raw_Data_End_Of_Session.sav](https://github.com/Sarahanc/LIS545-Data-Curation-Final-Project/blob/main/Raw_Data_End_Of_Session.sav)<br>
   This file contains participants anonymized identifiers, ages, genders, and the number associated with each participant's selected response for a given question. This table does not represent the actual questions the participants answered, nor the answers they chose. Rather, each question and answer is assigned an identifier.<br>
